# cli_weather
Check weather on CLI
Register an API Key at https://openweathermap.org

## Usage
```
npm install
npm link
```

## Commands
```
# Help & Commands
temp -h

# Version
temp -V

# API Key SubCommands
temp key set <API_KEY>
temp key get
temp key delete

# Check Mapping of Available citi to codes
temp check

# Check Temperature
temp check -c <code>
temp check -c <code> -f
```
